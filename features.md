# Future Features - Whis Chat Application

This document outlines potential features that may be added to Whis in future updates. These features are planned but not yet implemented.

## üîê Authentication & User Management

### User Accounts
- **User Registration & Login** - Email/password authentication with Firebase Auth
- **Social Login** - Sign in with Google, GitHub, Twitter, etc.
- **User Profiles** - Customizable profiles with avatars, bio, and status
- **Profile Pictures** - Upload and manage profile images
- **User Status** - Online, offline, away, busy status indicators
- **Account Settings** - Privacy settings, notification preferences, theme selection

### User Management
- **User Roles** - Admin, moderator, and member roles with different permissions
- **User Blocking** - Block users to prevent them from messaging you
- **User Muting** - Mute users to hide their messages without blocking
- **Friend System** - Add friends and see their online status
- **User Search** - Search for users by name or username

- **Message Threading** - Reply to specific messages with threaded conversations

- **Message Forwarding** - Forward messages to other rooms or users
- **Message Scheduling** - Schedule messages to be sent at a specific time
- **Read Receipts** - See who has read your messages (single/double check marks)
- **Message Quotes** - Quote and reply to previous messages inline
- **Message Reactions Count** - Show count of each reaction type
- **Message Search Filters** - Filter search by date, user, message type, etc.

### Rich Media
- **File Sharing** - Share documents, PDFs, videos, and other file types
- **Video Messages** - Record and send short video messages
- **Voice Messages** - Record and send audio messages
- **GIF Support** - Search and send GIFs from GIPHY or Tenor
- **Stickers** - Custom sticker packs and emoji stickers
- **Screen Sharing** - Share your screen during chat sessions
- **Location Sharing** - Share your location on a map
